# KF – 3D Landing Page

A modern landing page built with HTML, CSS.  
Features a 3D interactive model powered by Spline.

## Tech Stack

- HTML / CSS /
- [Spline](https://spline.design/) – 3D model & animation
- Custom local fonts (Poppins, Montserrat)

## Features

- Responsive layout
- 3D hero element (interactive)
- Custom typography
- Smooth animations

## Run locally

Just open `index.html` in your browser  
or use a local dev server (e.g. Live Server in VS Code).
